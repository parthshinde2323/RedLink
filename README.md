# 🩸 RedLink

> **Connecting Hospitals, Blood Banks & Donors — When Every Second Matters.**

RedLink is a real-time blood emergency coordination platform that connects **Hospitals, Blood Banks, and Donors** during critical situations. It leverages AI, real-time notifications, and live tracking to streamline emergency blood requests and improve response times, ultimately helping save lives.

---

## 🏆 Hackathon Achievement

🥈 **Runner-Up — Innoverse 2.0 Hackathon**

RedLink was developed during **TECHSPRINT Smart System Hackathon**, where it secured the **Runner-Up** position. The project was recognized for its innovative approach to emergency blood coordination by combining AI, real-time communication, and intelligent routing into a single platform.

---

## ✨ Features

* 🚨 Emergency Blood Request System
* 🔔 Firebase Cloud Messaging (FCM) emergency notifications with vibration alerts
* 🚑 Live Ambulance Route Tracking using Google Maps, OpenStreetMap & OSRM
* 🤖 AI Blood Demand Prediction using Groq (LLaMA 3)
* ✍️ AI SOS Message Generator for hospitals
* 🩸 Real-time Blood Inventory Management
* 👥 Multi-role system for Hospitals, Blood Banks & Donors
* 📍 Real-time location updates
* 🔐 Secure Firebase Authentication

## 📱 User Roles

### 🏥 Hospital

* Raise emergency blood requests
* Track ambulance movement
* View blood inventory
* Generate AI-powered SOS messages

### 🩸 Blood Bank

* Update blood stock
* Accept emergency requests
* Manage inventory

### ❤️ Donor

* Receive emergency alerts
* View nearby requests
* Respond to donation requests

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/RedLink.git
```

### Navigate to the Project

```bash
cd RedLink
```

### Install Dependencies

```bash
flutter pub get
```

### Run the Project

```bash
flutter run
```
